使用说明

在使用这个插件之前你需要引用几个文件。

1:jquery 1.5.1+
2:jquery.smartbox.js
3:jquery.smartbox.css

满足条件后就可以使用啦...

基本设置

此插件基本的设置，默认弹出层内容来自配置项。

  • 基本设置
  • HTML
  • JavaScript
标题

smartbox.js 是一款基于jquery 的开源的弹出层插件，它小巧但功能强大，允许用户在一个界面同时弹出多个对话框。弹出的内容支持多种形式，样式简洁多样。想要了解更多可以查看官方文档。 查看源码以及demo可以访问我的github。

底部
点我看效果 
<div id="basic" style="display: none;"></div>
<a id="basic_open" href="javascript:void(0);">点我看效果</a>
            
$(function(){
    $("#basic").smartbox({
        title:'标题',
        footer:'底部',
        width:500,
        height:300,
        content:'<p class="introduce_msg">smartbox.js 是一款基于jquery 的<span style="color: red;">开源的</span>弹出层插件，它小巧但功能强大，允许用户在一个界面同时弹出多个对话框。弹出的内容支持多种形式，样式简洁多样。想要了解更多可以查看<a target="_blank" href="http://smartbox.huangsw.com/cn/introduce.html">官方文档</a>。 查看源码以及demo可以访问<a href="https://github.com/huangshuwei/jquery.smartbox" target="_blank">我的github</a>。</p>',
        isAutoShow:false
    });

    $("#basic_open").click(function () {
        $("#basic").smartbox("open");
    });
})

异步请求html

此插件支持异步获取html内容。

  • 异步请求html
  • HTML

smartbox 简介-通过ajax
点我看效果 
<div id="ajaxRequestHtml" style="display: none;"></div>
<a id="ajaxRequestHtml_close" href="javascript:void(0);">点我看效果</a>
$(function(){
        $("#ajaxRequestHtml").smartbox({
            type:'option',// 'html':title、content、footer 内容来自html；'option':title、content、footer 内容来自配置 |type string
            ajaxSetting:{
                contentType:'html', // 'html':异步加载html；'img':异步加载图片；'iframe':异步加载iframe（可以解决跨域问题） |type:string
                url : '../demo/html/introduce.html',
                isShowLoading : true,// 是否显示加载效果 |type:bool
                loadingType:'text',// 'img':加载中以图片的效果展示；'text':加载中以文字的形式展示 |type:string
                loadingText:'<span style="color: #6f6f6f;">页面加载中...</span>',
                errorContent:'<span style="color:red;">请关闭弹窗重新再试！</span>' // 请求错误时显示的内容 |type:html
            },
            title:'smartbox 简介-通过ajax',
            width:500,
            height:330,
            isAutoShow:false,
            isShowFooter:false
        });

        $("#ajaxRequestHtml_close").click(function () {
            $("#ajaxRequestHtml").smartbox('open');
        });
    })

多窗口模式（不同级）

此插件不仅支持多窗口同级模式，并且支持不同级的形式。要求条件是设置‘zIndex’,‘zIndex’值大的在上方，
并且要求设置遮罩层即‘isShowOverlay:true’，因为不同级模式不适用遮罩层就没有意义了。
